# playmind-py

playmind-py is a Python 3 command line game to practice your Math skills.

Train your mind doing mental calculations.

## Operations supported: 
* Addition.
* Subtraction.
* Multiplication.
* Division.

## Clone the game:
``` git clone git@github.com:ronnyml/playmind-py.git ```

## Run the game:
``` ./playmind.py ```
